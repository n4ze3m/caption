### Installation ðŸ”§


1. Clone the repo
   ```sh
   git clone https://github.com/n4ze3m/caption.git
   ```

2. Install pnpm packages
   ```sh
    pnpm install
    ```

3. Create a .env file in the root directory and add the following

    ```sh
    FIREWORKS_API_KEY=YOUR_API_KEY
    ```

4. Run the project
    ```sh
    pnpm run dev
    ```
